프로젝트 개요

유튜브, 인스타그램 등 다양한 소셜 미디어 플랫폼에서 30분 단위로 조회수 데이터를 수집하고 분석하는 시스템을 구축했습니다. 각 플랫폼별 API를 활용하여 정기적으로 데이터를 수집하고, 트렌드 분석과 인사이트 도출을 통해 콘텐츠 전략 수립을 지원합니다.

기술 스택

Python Pandas NumPy Matplotlib MySQL

주요 기능

  • 다중 플랫폼 데이터 수집
  • 30분 단위 조회수 집계
  • 플랫폼별 비교 분석
  • 트렌드 분석 및 시각화
  • 자동화된 데이터 수집
  • 인사이트 리포트 생성

성과 및 결과

  • 3개 플랫폼 동시 데이터 수집
  • 30분 단위 정확한 집계
  • 플랫폼별 트렌드 분석
  • 콘텐츠 전략 수립 지원
