프로젝트 개요

KiyoungFnB 브랜드 전략본부 파트너십팀에서 개발한 스타트업 창업 지원 전용 GIS 상권분석시스템입니다. 배달의민족 음식점 순위 연동을 통해 실제 매장 정보와 검증된 데이터를 활용하며, 세대수 분석과 유동인구 밀도 분석으로 정확한 상권 평가를 제공합니다. 4단계 개발 계획을 통해 점진적으로 기능을 확장하여 창업자들에게 최적의 상권 분석 솔루션을 제공합니다.

기술 스택

GIS JavaScript Python Google Maps API 배달의민족 API 웹 시각화

주요 기능

  • 스타트업 지원 시스템: 창업자 대상 맞춤형 상권분석 대시보드
  • 배달의민족 연동: 실시간 음식점 순위 및 매장 정보 수집
  • 세대수 분석: 지역별 아파트, 빌라, 오피스텔, 주택 현황
  • 유동인구 분석: 시간대별, 요일별, 연령별 유동인구 밀도 시각화
  • GIS 기반 지도: 인터랙티브 상권 지도 및 분석 원형 표시
  • CSV 다운로드: 분석 데이터 추출 및 리포트 생성

성과 및 결과

  • 4개 이미지 갤러리: 포괄적인 시스템 기능 시각적 표현
  • 스타트업 맞춤 서비스: 창업자 대상 특화 상권분석 플랫폼
  • 실시간 데이터 연결: 배달의민족 API 연동으로 검증된 매장 정보
  • 다층 분석 기능: 인구, 유동인구, 매출 정보 통합 분석
  • 모바일 최적화: 터치 인터페이스 및 자동 카테고리 선택
  • 빅데이터 기반 평가: A등급 상권 가이드 기능 도입 예정
