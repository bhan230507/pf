프로젝트 개요

다양한 데이터 소스에서 실시간으로 데이터를 수집하고, 정제하여 분석 가능한 형태로 변환하는 시스템을 개발했습니다. Apache Airflow를 활용한 워크플로우 자동화와 모니터링 시스템을 구축하여 데이터 파이프라인의 안정성과 효율성을 확보했습니다.

기술 스택

Python Apache Airflow MySQL Redis Docker Kubernetes

  • 분산 처리 시스템 구축
  • 실시간 모니터링 대시보드
