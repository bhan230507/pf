프로젝트 개요

KiyoungFnB 기업의 전체 데이터 생태계를 통합하는 ETL 파이프라인을 구축했습니다. Google Cloud Platform을 기반으로 Jenkins CI/CD, Apache Airflow, BigQuery 등을 활용하여 대용량 데이터 처리와 실시간 분석이 가능한 확장 가능한 시스템을 구현했습니다. 더불어 Coupang, Baemin, NaverMap 등 다양한 모바일 앱과 연동하여 폭넓은 데이터 수집 범위를 확보했습니다.

기술 스택

Python Django Jenkins Nginx Gunicorn Apache Airflow Google Cloud Platform BigQuery Google Cloud Storage MySQL PostgreSQL

주요 기능

  • GCP 기반 클라우드 인프라 구축
  • Jenkins CI/CD 파이프라인 자동화
  • 다중 데이터 소스 통합 관리 (API, Files, DB, Other Cloud)
  • 모바일 앱 연동 시스템 (Coupang, Baemin, NaverMap)
  • KiyoungFnB 내부 시스템 통합
  • 다층 서버 환경 관리 (Master-Slave 구조)
  • BigQuery를 활용한 빅데이터 처리
  • Google Cloud Storage 기반 데이터 보관
  • 실시간 데이터 변환 및 파이프라인 오케스트레이션
  • BI 도구 연동 (Looker, Google Sheets)

성과 및 결과

  • 전사 데이터 파이프라인 통합으로 처리 효율성 500% 향상
  • 다중 플랫폼 연동을 통한 데이터 수집 범위 10배 확대
  • GCP 클라우드 인프라로 시스템 확장성 및 안정성 확보
  • BigQuery 활용으로 대용량 데이터 처리 속도 700% 개선
  • Jenkins CI/CD 도입으로 배포 자동화 및 오류율 90% 감소
  • 모바일 앱 연동으로 실시간 데이터 활용도 증가
  • 통합 BI 시스템 구축으로 의사결정 속도 향상
  • 운영 비용 최적화로 IT 예산 45% 절감
