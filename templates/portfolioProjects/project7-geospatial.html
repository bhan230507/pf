프로젝트 개요

KiyoungFnB에서 직접 개발한 스타트업 창업 지원 전용 GIS 상권분석시스템입니다. 배달의민족 음식점 순위 연동을 통해 실제 매장 정보와 검증된 데이터를 활용하며, 세대수 분석과 유동인구 밀도 분석으로 정확한 상권 평가를 제공합니다. 스타트업 창업 지원을 목적으로 개발되어 창업자들에게 최적의 상권 분석 솔루션을 제공합니다.

기술 스택

GIS (Geographic Information System) JavaScript Python Folium Google Maps API 배달의민족 API 웹 지도 시각화 데이터 분석 상권 분석

주요 기능

  • 스타트업 창업 지원: 창업자 대상 맞춤형 상권분석 및 입지 추천 시스템
  • 배달의민족 API 연동: 실시간 음식점 순위 및 매장 정보 수집 및 분석
  • 세대수 데이터베이스: 지역별 아파트, 빌라, 오피스텔, 주택 현황 분석
  • 유동인구 분석: 시간대별, 요일별, 연령별 유동인구 밀도 시각화
  • GIS 지도 인터페이스: 인터랙티브 상권 지도 및 분석 원형 표시
  • 데이터 내보내기: CSV 다운로드 기능으로 분석 리포트 생성
  • 실시간 데이터 수집: 다양한 API를 통한 최신 데이터 자동 갱신
  • 모바일 대응: 터치 인터페이스 및 반응형 웹 디자인

성과 및 결과

  • 창업 지원 플랫폼 구축: 직접 개발한 스타트업 전용 상권분석 시스템
  • 다각도 데이터 통합: 배달의민족 API, 세대수 데이터, 유동인구 정보를 통합한 종합 분석
  • 실시간 데이터 처리: API 연동을 통한 실시간 매장 정보 수집 및 분석 자동화
  • 공간 분석 솔루션: GIS 기술을 활용한 지리적 데이터 시각화 및 인터랙티브 지도 구현사용자 친화적 인터페이스: 직관적인 웹 인터페이스로 쉽게 접근 가능한 상권 분석 도구
  • 데이터 중심 의사결정: 객관적 데이터를 바탕으로 한 창업 입지 선택 지원
  • 확장성 설계: 향후 추가 기능 확장을 고려한 모듈식 아키텍처 구현
  • 실용적 도구 제공: 창업자들이 실제로 활용할 수 있는 실용적인 분석 플랫폼
