프로젝트 개요

다양한 소스에서 키워드를 실시간으로 수집하고 분석하여 트렌드와 인사이트를 제공하는 모니터링 시스템을 개발했습니다. 알림 기능과 리포트 자동 생성을 통해 효율적인 키워드 관리가 가능하며, 마케팅 전략 수립에 활용됩니다.

기술 스택

Python Selenium BeautifulSoup Django Redis FastAPI

주요 기능

  • 실시간 키워드 모니터링
  • 다양한 소스 데이터 수집
  • 트렌드 분석 및 예측
  • 알림 시스템 및 경고
  • 자동 리포트 생성
  • 데이터 시각화

성과 및 결과

  • 트렌드 감지 시간 80% 단축
