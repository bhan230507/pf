프로젝트 개요

Agent Development Kit (ADK)를 활용하여 배달의민족 데이터 분석 AI 에이전트를 개발했습니다. 에이전트가 자율적으로 웹 크롤링, 데이터 수집, 분석, 시각화까지 수행하며, 외식업계의 시장 동향과 경쟁 정보를 실시간으로 모니터링하여 전략적 의사결정을 지원합니다.

기술 스택

Agent Development Kit (ADK) AI Agent Framework Python Pandas NumPy Matplotlib Selenium BeautifulSoup Automated Data Collection

주요 기능

  • AI 에이전트 자동화: ADK 에이전트가 배민 데이터 수집부터 분석까지 전 과정 자동화
  • 지능형 웹 크롤링: 에이전트가 배민 사이트를 자동 탐색하며 실시간 데이터 수집
  • 자율적 데이터 분석: 수집된 데이터를 AI 에이전트가 자동으로 분석하고 인사이트 도출
  • 경쟁 업체 동향 추적: 식당 순위, 메뉴 변화, 가격 변동 등 실시간 모니터링
  • 시장 트렌드 예측: 머신러닝을 활용한 트렌드 패턴 분석 및 예측 모델링
  • 자동 리포트 생성: 분석 결과를 자동으로 시각화하고 리포트 생성
  • 실시간 알림 시스템: 중요한 시장 변화 감지 시 자동 알림 발송

성과 및 결과

  • 에이전트 자동화 효율성: 기존 수동 분석 대비 시장 분석 시간 80% 단축

  • 자율 분석 성능: 에이전트의 자율적 분석으로 인간 수준의 인사이트 도출 가능
  • 실시간 모니터링 구현: 24/7 에이전트 가동으로 배민 시장 변화 실시간 감지
