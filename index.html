성격 및 강.약점

저는 체계적이고 분석적인 성격으로, 복잡한 데이터를 정리하고 패턴을 찾아내는 것을 즐깁니다. 특히 문제를 발견하면 근본 원인을 파악하고 체계적인 해결책을 제시하는 것을 강점으로 가지고 있습니다.
약점으로는 때로는 너무 많은 기술을 동시에 학습하려는 경향이 있어 이를 극복하기 위해 우선순위를 정하고 단계별로 깊이 있게 학습하는 방식을 채택하고 있습니다.

직무수행 역량 및 자세

현재 주식회사기영에프앤비에서 데이터분석팀 팀원(과장)으로 근무하며, Apache Airflow를 활용한 ETL 자동화 시스템 구축, GCP 기반 클라우드 인프라 구축, Django를 활용한 데이터 시각화 대시보드 개발 등 빅데이터 파이프라인 구축 및 운영의 전 과정을 담당하고 있습니다.
특히 대용량 데이터 처리 성능을 40% 향상시키고, 데이터 처리 시간을 80% 단축하는 등 구체적인 성과를 달성하며 데이터 기반 의사결정을 지원하고 있습니다.

목표

Python + Django + Apache Airflow + GCP 기반의 차세대 데이터 플랫폼을 설계하여 실시간 데이터 처리를 넘어선 AI 기반 예측 분석 시스템을 구축하겠습니다. 현재 달성한 데이터 처리 성능 40% 향상을 넘어서, ElasticSearch와 머신러닝을 결합한 지능형 분석 엔진을 개발하여 텍스트 데이터부터 구조화된 데이터까지 통합 분석이 가능한 플랫폼을 만들겠습니다.

특히 기존 MySQL, Redis, PostgreSQL 활용 경험을 바탕으로 올 하이퍼브리드 데이터 웨어하우스를 설계하여, 대용량 데이터의 실시간 처리와 장기 보존을 동시에 가능하게 하고, D3.js 시각화 기술과 결합하여 데이터 인사이트를 직관적으로 전달할 수 있는 대시보드 생태계를 구축하겠습니다.

궁극적으로는 현재 포트에서의 ETL 파이프라인 운영 경험을 바탕으로 클라우드 네이티브 데이터 아키텍처 전문가 (Cloud-Native Data Architecture Expert)가 되어 차세대 데이터 인프라의 표준을 만들어가는 것이 목표입니다.
